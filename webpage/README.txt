Fast Food Runner:
A web based application that will calculate the maximum quantity of a certain food characteristic specified by the user 
while maintaing within the price range which the usser also specified. The goal of the application is to be humurous in that
we will search for a fast food restaurant but tell the user to walk or run a certain distance so that they will brun off
the calories they are about to ingest.
